# Walid Birouk's Portfolio

Welcome to the source code repository for my personal portfolio website. This portfolio showcases my skills, projects, and provides a comprehensive overview of my background and experiences in the field of AI and web development.

## Overview

The portfolio is built with [Gatsby](https://www.gatsbyjs.com/), a modern web framework for React, and features a home page with a hero image, navigation buttons, an "About Me" section styled like a CV, and a projects page that displays project cards. Clicking on a project card redirects to a detailed project page with images and a link to the GitHub repository.

### Features

- **Home Page**: Introduction with a hero image and navigation buttons for ease of access.
- **About Me**: Detailed section resembling a CV, highlighting my skills, education, and experiences.
- **Projects Page**: Showcases my projects with titles, technologies used, and detailed project information upon selection.
- **Responsive Design**: The site is fully responsive, ensuring a seamless experience across all devices.

### Technologies Used

- Gatsby
- React
- Tailwind CSS
- GraphQL
- Gatsby Plugins (gatsby-plugin-image, gatsby-plugin-sharp, gatsby-transformer-sharp, etc.)

## Getting Started

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/Walid-Birouk/portfolio.git
   ```
2. Install NPM packages:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   gatsby develop
   ```

## Deployment

To build the project for production, run:

```sh
npm run build
```

And to deploy it on GitHub Pages:

```sh
npm run deploy
```
